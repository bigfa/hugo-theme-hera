# Hugo Theme Hera

![](https://static.fatesinger.com/2024/12/2u80bhyxkmru4o9j.png)

## How to use

```
git submodule add https://github.com/bigfa/hugo-theme-hera.git themes/hera

echo "theme = 'hera'" >> config.toml
```
